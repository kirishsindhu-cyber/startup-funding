# 🚀 Startup Funding Analysis

> **Activity 27 — Startup Funding Analysis**  
> Big Data Activity-Based Learning Model

An interactive dashboard built with **Python + Streamlit** that analyzes startup funding trends including top funded sectors, investor activity, and year-wise growth.

---

## 📊 Dashboard Features

| Task | Description |
|------|-------------|
| Top Funded Sectors | Bar chart of sectors by total funding & deal count |
| Investor Trends | Most active investors + investment type distribution |
| Year-wise Growth | Line chart of deals & bar chart of funding over years |
| KPI Summary | Total startups, funding, top sector, top investor |

---

## 🗂 Dataset

**Source:** [Indian Startup Funding — Kaggle](https://www.kaggle.com/datasets/sudalairajkumar/indian-startup-funding)

Download `startup_funding.csv` from Kaggle and upload it in the app sidebar.

---

## 🛠 Tech Stack

- **Python 3.x**
- **Pandas** – data cleaning and analysis
- **Matplotlib / Seaborn** – visualizations
- **Streamlit** – interactive web dashboard

---

## ⚙️ Setup & Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/startup-funding.git
cd startup-funding

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```
